Web Dizayn dilləri

Web səhifələrinin front-end, yəni ön hissəsinin yaradılmasında istifadə edilən, əsas misyaları dizayn olan dillərdir. Bütün web səhifələrinin dizaynında bunlardan istifadə olunur. Front-end dizaynında istifadə edilən, eyni zamanda bir o qədər əsas və təməl olan HTML, CSS, JAVASCRIPT-dir. Geriyə qalanlar isə bu dillərin kitabxanalarıdır. Back-end hissəsinə aid dərsləri isə Server bölümündədir.

PS: Web dizaynında istifadə edilən bu dillər, proqramlaşdırma dili sayılmır və proqramlaşdırma dilləri ilə qarışdırlmamalıdır!





Proqramlaşdırma dilləri

Proqramlaşdırma dilləri vasitəsi ilə istənilən platformalar üçün Mobil, Desktop və.s istənilən proqramlar və oyunlar yazmaq mümkündü. Hər proqramlaşdırma dilinin özünə məxsus yazılışı, kodlanması olduğu kimi eyni zamanda bir-birlərinə oxşar cəhətləridə çoxdu.
