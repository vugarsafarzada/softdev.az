IDLE nədir?


Bundan əvvəlki dərsimizdə biz Pythonu kompüterimzə yükləmiş və Əmr sətiri üzərində, yəni cmd.exe üzərində işə salmışdıq.



Bu o deməkdir ki biz Python'un əmr panelini, kompüterimzin cmd.exe proqramı üzərində açdıq. Yəni əslində Python da özünə məxsus bir əmr sətiri var. Əgər istəsək Python'un quraşdırdıqdan sonra onu start menusunda tapa bilərik.





Gördüyünüz bu "IDLE" adlı proqram, Python üçün Comand Line, yəni əmr sətiri funksiyasını yerinə yetirir və Python ilə birlikdə kompüterimzə yüklənir. Biz Python əmrlərimizi, Python kodalarımızı burda istifadə edə bilərik.


Hello World!


Proqramlaşdırma dünyasında belə bir adət vardır, hansı ki bir proqramlaşdırma dilini öyrənməyə başladığında ilk proqram olaraq, ilk öyrənilən kod olaraq həmişə ekrana HELLO WORLD yazdırılar. Gəlin bizdə ilk kodumuzu yazaraq Hello World deyək. Bunun üçün bizə lazım olan Python'un "print()" əmridir. Bu isə öyrəndiyimiz ilk Python kodu oldu :)




PyCharm nədir?


Pythonun əmr sətirində kod yazmaq biraz zəhmətli olduğu üçün və fayılların idarəsinində bir o qədər çətin olduğu üçün, işimizi asanlaşdırsın deyə hər proqramlaşdırma dilində olduğu kimi Python'da da IDE(Integrated development environment) deyilən fərqli növ köməkçi tətbiqlərdən istifadə edilir. PyCharm'da bu tətbiqlərdən biridir. PyCharmı öz veb səhifəsindən yendirib kompüterimzə quraşdıra bilərik.
PyCharmı kompüterinizə yükləmək üçün bura daxil olun!
