HTML Nədir?

HTML Veb səhifələr yaratmaq üçün standart işarələmə dilidir.

  • HTML → Hyper Text Markup Language → "Hiper mətn işarələmə dili" deməkdir.
  • HTML bir veb səhifəsinin quruluşunu təsvir edir.
  • HTML brauzerlərə(Chrome, FireFox, Opera və s.) məzmunun necə göstəriləcəyini izah edir.
  • HTML elementləri "teqlər" ilə təmsil olunur.
  • Brauzerlər HTML teqlərini göstərməz, ancaq səhifənin məzmununu göstərmək üçün onlardan istifadə edərlər.


HTML Teqlər(Etiketlər) nədir?

HTML kodları teqlər ilə yazılır (Teq→ingilis dilindəki "tag" yəni "etiket" deməkdir). Bir teq kiçikdir (<), böyükdür(>) işarəsi içində yazılan bir HTML elementindən ibarətdir. Burda element yəni bir əmir, bir kod deməkdir. HTML teqləri quruluşuna görə iki növə ayrılır:
"açılıb-bağlanan" və "açılıb-bağlanmyan". HTML'də əksər teqlər açılıb-bağlanan növdədir. İndi isə gəlin bir teq nümunəsinə baxaq:

My cat is very grumpy
